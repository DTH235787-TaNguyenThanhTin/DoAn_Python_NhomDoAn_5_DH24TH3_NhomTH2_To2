## Thêm ảnh trang chủ

Để hiển thị ảnh ở trang chủ, đặt tệp ảnh vào thư mục `assets` với một trong các tên sau:

- `assets/hero.jpg` (ưu tiên) hoặc
- `assets/hero.png`

Nếu bạn muốn dùng ảnh đính kèm trong cuộc trò chuyện này, hãy tải ảnh xuống máy của bạn rồi sao chép/dán vào `assets/hero.jpg`.

Ví dụ tạo thư mục và chạy ứng dụng (PowerShell):

```powershell
mkdir .\assets ; python -m pip install pillow  # pillow chỉ cần nếu dùng JPEG
python .\main.py
```

Nếu bạn muốn, tôi có thể tạo file `assets/hero.jpg` trong repo nếu bạn tải ảnh trực tiếp vào workspace hoặc cung cấp một URL công khai tới ảnh.
